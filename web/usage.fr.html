utilisation de rpmrebuild

voici quelques utilisations basiques de rpmrebuild

reconstruire un package rpm depuis un package installé

vous ne trouvez plus le fichier rpm que vous avez utilisé ?
re-faites-le avec la commande rpmrebuild nom_du_package

changements de fichiers

Vous avez modifié un fichier de configuration et vous voulez distribuer votre version ?
utilisez la même commande : rpmrebuild nom_du_package, parce que rpmrebuild fabrique le package à partir des fichiers installés

ajouter ou supprimer un fichier

Vous devez connaitre un minimum la syntaxe des fichiers "spec" (consulter le RPM-HOWTO, ou rpm.org).
Ajouter le fichier ou supprimez le dans le repertoire désiré
lancez la commande rpmrebuild -e nom_du_package. rpmrebuild va ouvrir dans votre éditeur favori le fichier "spec", ce qui vous permettra de modifier les fichiers de la section "%files"
