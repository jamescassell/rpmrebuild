installation de rpmrebuild

Comme rpmrebuild travaille sur les packages rpm, il est fourni naturellement sous ce format.

quel package choisir ?

  • rpm4.noarch.rpm : pour les distributions "modernes", qui utilisent rpm version 4.x
  • rpm3.noarch.rpm : pour les "vieilles" distributions, qui utilisent encore rpm en version 3.x

vérifier le package

Tous les packages rpm sont signés avec ma clef gpg. Je vous recommande fortement de vérifier tout package téléchargé avec la commande rpm -K rpmrebuild*.rpm.

installation

Rpmrebuild est codé en shell (bash), donc il ne dépends pas de librairies, ni de l'architecture. Il s'installe avec la commande : rpm -Uvh rpmrebuild-{version}.noarch.rpm.

en cas de problème

  • vous pouvez essayer de fabriquer votre propre package rpm à partir du fichier source rpmrebuild-{version}.src.rpm, avec la commande rpmbuild --rebuild rpmrebuild-{version}.src.rpm, puis installer votre fichier rpm
  • envoyer un rapport de bug
